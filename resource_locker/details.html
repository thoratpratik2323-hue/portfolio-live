IP VERSE

High-Fidelity Resource Directory Clone

About The Project

IP Verse is a precision-engineered clone of the popular 'WPLocker' resource directory. It replicates the complex grid algorithms, typography, and structural hierarchy of the original site while introducing a modern, ad-free experience with a custom 'High-Voltage' Dark UI.

HTML5 Structure CSS3 Variables Vanilla JS Live DOM Manipulation

Key Features

Pixel-Perfect Clone

Recreated the exact DOM structure and CSS layout of the source material for 1:1 visual fidelity.

Neon Dark Mode

A custom-built dark theme with high-contrast text and neon green accents for a cyberpunk aesthetic.

Live Search

Instant client-side filtering system that searches through resource cards in real-time, removing the need for page reloads.

Ad-Block Core

Optimized rendering path that strips tracking scripts and ad iframes for a superior reading experience.
